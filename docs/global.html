getCookies(browser, limitedopt, cookiesFilePathopt) → {Array.<object>}

Returns saved cookies from a specified Chromium web browser.
Parameters:
Name Type Attributes Description
browser string The browser to retrieve cookies from, keep in mind that the browser must use the Chromium storage facility.
limited boolean <optional>
 Whether or not to return only important information.
cookiesFilePath string <optional>
 Path for cookies file, only necessary if your browser's cookies file does not reside in its default location.
